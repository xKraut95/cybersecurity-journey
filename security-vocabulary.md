# Security vocabulary & fundamentals

This document contains core security concepts learned during my cybersecurity training and studies.

## Firewall
A firewall controls network traffic based on predefined rules to allow or block communications, it is important tool to prevent unauthorized devices from accesing corporate networks

## Network Access Control (NAC)
NAC is a virtual machine or device that controls the access of a network, it has a complete visibility of network profiles. It's a centralized machine that takes care of the corporate network.

## Sandbox
Sandbox is a system who confines the actions of an application in a secure virtual enviroment. It is very useful to zero-day attacks and improving the security in the network.

## Malicious Actor
Is a person or a group of persons with different motivations, personalities and attack methods, they are not homogeneous. They can be anyone, for example, hacking groups or individual hacktivists.

## CIA Triad

- The special triad for cybersecurity analists

- Confidentiality, the data must be accesed by authorized people and restricted to anyone else

- Integrity, the data must remain authentic and it mustn't be altered

- Availability, the data must be available for all the authorized persons.


## DAD triad

- The triad that all cybersecurity analists must avoid, it means

- Divulgation: The data is exposed to unauthorized people

- Alteration: The data is altered nor authentic.

- Denegation: The authorized agents aren't allowed to enter the data.
